Current Projects

  1. Graph Gaussian Process with linearly constrained kernels learned from data

    Collaboration with Alexander Terenin, Haolin Yu, Pascal Poupart
    Let us consider the task of predicting real-valued function on nodes of a graph given samples of this function on certain nodes. For example, the graph can be a road network, and we would like to predict traffic speed at any point in the network when given speed sensor data at certain locations. In this project, we want to make this prediction more accurate and data-efficient by using Gaussian process regression with a kernel that encodes prior information about the structure of the function. In particular, we are concerned with the setting where the function satisfies some linear structure. In a previous work, we have shown that by using manually crafted prior information, this approach indeed leads to superior performance. Currently, we are working on learning this prior information directly from data.

  2. Denoising time-series gene regulatory networks

    Collaboration with Sheng Wang, Saurabh Sinha
    Gene regulatory networks (GRNs) describe how gene expressions are regulated, the study of which is crucial to understanding a wide range of biological processes. GRNs can be reconstructed from transcriptomics and chromatin accessibility data via mathematical modelings or machine learning models. Most existing methods reconstruct GRNs at a single time point. However, for time-series data (e.g. in developmental biology) it is important to take into account the temporal correlation between time points. We are applying generative deep learning models, in particular conditional variational autoencoder and transformer architecture, to produce holistically denoised GRNs for time series data. Preliminary results show that generative models strongly outperform the best baseline (plots coming soon!).

  3. Foundation models for practical bandit problems

    Group project
    We are developing a foundaton model for contextual bandits, with an encoder that maps inputs from different domains into a common reference space.
